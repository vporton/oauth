    Function OAuthSession.load

    Load OAuthSession data from Vibe.d session.

    static OAuthSession load (
      immutable(OAuthSettings) settings,
      scope vibe.http.session.Session httpSession
    ) @safe;

    Parameters

    NameDescription
    settings The OAuth settings to use.
    httpSession The Vibe.d session.

    Throws

    OAuthException if the session data as found in the Vibe.d session appears corrupt or is not valid for the provider referenced by settings.
